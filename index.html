Tags

As tags são comandos que nos permite manupilar a estrutura do html e elementos do nosso site. cada tag tem uma determinada função, Por exemplo, a tag "Title" que nos permite dar um nome para nossa pagina que é exibido na aba do navegador.

Atributos das Tags

Atributos são alguns tipos de comando que existem para manipularmos ou ajustar alguma tag, as tags podem ter atributos similares, mas existem atributos que pertecence a somente um tipo de tag.

Textos

As tags nos ajudam a manipular nosso texto, com as tags podemo deixar nosso texto em negrito, em italico, sublinhado, ou ate mesmo deixar ele marcado como um marcador de texto. essas ferramantas são muito ultilizadas na hora da criaçao de um artigo ou na criaçao de um blog. Existem tags que manipula a estrutura do texto podendo adcionar titulos, subtitulos, paragrafos, citações, etc...

Listas

Dentro do html existem 2 tipos de listas, as lista ordenadas e as lista não ordenadas. A lista ordenada são ordenadas por números, ja as listas não ordenadas são separdas por tópicos ou seja não temos uma ordem especifica para esse tipo de lista. Na maioria dos casos nos utilizamos esses tipos de listas para criarmos menus para o nosso site.

Links

Para navegar na internet utilizamos links para trocas rapidas de paginas ou abrir uma outra paginas em uma outra aba do nosso navegador para isso usamos a tag "a" que significa ancora e logo apos declaramos um atributo chamado href para que possamos passar a referencia da onde devemos ir apos o click do mouse.
